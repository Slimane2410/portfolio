RETURN TO MATRIX

Smart Workshop - Data & AI

Local Text-to-SQL system for a FabLab environment, giving non-technical users safe access to workshop data.

Mission Overview

This project builds an end-to-end data and AI pipeline for a smart workshop. It centralizes fragmented technical information (machines, parameters, tools, documentation) into a structured SQLite database and exposes it through natural-language queries.

Objective: deliver a local, explainable Text-to-SQL system with strict security layers, traceability, and a minimal Streamlit interface for real-world usage.
